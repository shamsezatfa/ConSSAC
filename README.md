# ConSSAC: Conservative Sigmoidal Soft Actor-Critic for Offline Dialogue Policy Optimization

This repository contains an implementation of ConSSAC, as described in the corresponding paper, integrated with Convlab-3.

ConSSAC is an off-policy deep reinforcement learning method designed for task-oriented dialogue policy learning. It aims to enhance the stability, exploration, and robustness of dialogue policies in discrete task-oriented dialogue environments.

## Features

*   Implementation of ConSSAC for task-oriented dialogue policy learning.
*   Integration with Convlab-3 for experimentation and evaluation.
*   Utilization of the ConSSAC approach to alleviate underestimation bias and distribution shift.

## Requirements

*   Convlab-3

## Usage

1.  Clone this repository.
2.  Install the required dependencies of Convlab-3.
3.  Follow the instructions provided in the documentation or examples to train and evaluate the ConSSAC model.
